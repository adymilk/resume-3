### :world_map: Vue高德地图

[【预览】](https://leachzhou.github.io/vue-amap-demo) 
[【源码】](https://github.com/LeachZhou/vue-amap-demo)

项目介绍：

基于vue高德地图的一个操作demo，包括输入自动提示、点击获取地址存入输入框等功能。
